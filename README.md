Jiji Web Scraper

This project scrapes product listings from Jiji, a popular online marketplace. The goal is to extract key information such as:
Imagelink ,Product_name	,Price	Product_condition	Product_verification
After scraping, the data is converted into a Pandas DataFrame for easy analysis and further processing.
Tools & Libraries Used
Python– main programming language
Requests – to send HTTP requests and fetch web pages
BeautifulSoup – to parse HTML and extract information
Pandas – to store and manipulate scraped data as a DataFrame
Jupyter Notebook – to develop and document the scraping process
